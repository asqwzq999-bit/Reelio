Cara menyimpan di masing-masing perangkat

💻 Menyimpan di PC

Di browser desktop atau laptop, ikuti langkah berikut.

  1. Buka situs TikTok di browser Anda.
  2. Klik ikon bagikan pada video dan pilih "Salin link."
  3. Tempel link ke kotak di halaman ini lalu tekan unduh.
  4. Klik tombol format (MP4/MP3) yang Anda inginkan di layar hasil.

📱 Menyimpan di iPhone

Bisa langsung dipakai di Safari — tanpa perlu aplikasi tambahan.

  1. Buka video di aplikasi TikTok, ketuk Bagikan → Salin link.
  2. Pindah ke Safari, buka halaman ini, dan tempel link-nya.
  3. Ketuk unduh, lalu ketuk tombol MP4.
  4. File akan muncul di folder Unduhan pada aplikasi Files.

🤖 Menyimpan di Android

Bekerja dengan cara yang sama di Chrome atau browser default Android lainnya.

  1. Di aplikasi TikTok, ketuk Bagikan → Salin link.
  2. Buka halaman ini di browser dan tempel link-nya.
  3. Ketuk unduh, lalu ketuk tombol MP4.
  4. File akan otomatis tersimpan ke folder Unduhan perangkat Anda.

MP4 atau MP3, mana yang harus dipilih?

Jika berencana menyimpan videonya sendiri atau memakainya untuk editan lanjutan, pilih MP4. Jika hanya butuh musik latar atau suara, MP3 lebih cocok.

  • MP4 — File asli lengkap dengan gambar dan suara, siap dipotong dan disusun ulang
  • MP3 — Ukuran jauh lebih kecil, siap dipakai untuk didengarkan atau audio editing

Belum yakin butuh yang mana? Unduh MP4 dulu lalu ekstrak audionya kapan pun diperlukan.
